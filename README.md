# Install Streamlit
**simple**
```
pip install streamlit
```
**with venv**
```
== create folder ==
mkdir project-numtod

== open folder ==
cd project-numtod

== create venv ==
python -m venv .venv

== activate environment ==
.venv\Scripts\activate.bat

= install streamlit =
pip install streamlit

== deactivate environment ==
deactivate
```

# Run App
```
streamlit run app.py
```
